# SchoolApp
Clean Architecture App with Jetpack Compose

App use:
* Dependency injection with Dagger-Hilt
* Clean Architecture, with data, domain and presentation layer
* Retrofit for API call
* Pull to refresh
* Local Caching with Room, works offline
